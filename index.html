HTML

  1. HTML was developed by Tim-Berners Lee in 1996. An HTML plain file contains only text and nothing else. HTML stands for Hyper Text Markup Language.
  2. Hypertext is simply a piece of text that works as a link
  3. Markup language is way of writing layout information within documents.

Tags

1. Container Tags
Container tags are used to wrap content in a container. They have an opening tag and a closing tag. For example, <div> and </div>
2. Empty Tags
Empty tags are used to create content that doesn't need a closing tag. For example, <br> and <hr>

Structure of HTML Document

    
      <html>
        <head>
          <title>Title of the document</title>
        </head>
        <body>
          <h1>This is a heading</h1>
          <p>This is a paragraph</p>
        </body>
      </html>

Body Tag

The body tag is used to define the main content of the HTML document. It contains all the content that is visible to the user. The body tag is placed inside the HTML tag. The body tag has an opening tag <body> and a closing tag </body>.

Attributes of Body tags are,

  • background="url": Specifies the background image for the document.
  • bgcolor="color": Sets the background color of the document.
  • text="color": Sets the color of the text in the document.
  • link="color": Sets the color of unvisited links.
  • vlink="color": Sets the color of visited links.
  • alink="color": Sets the color of active links.
  • leftmargin="value": Sets the left margin of the document.
  • rightmargin="value": Sets the right margin of the document.

Heading Tags

Heading tags are used to define headings in an HTML document. There are six levels of headings, from <h1> to <h6>, with <h1> being the highest level and <h6> the lowest.

Syntax:

<h1 align="center">This is a heading</h1>

Attributes of Heading tags are,

  • align="left|center|right": Specifies the alignment of the heading.

Horizontal Rule Tag

The horizontal rule tag is used to create a horizontal line in an HTML document. It is an empty tag and does not require a closing tag.

Syntax:

<hr align="center" noshade size="3" width="50%" color="blue">

Attributes of Horizontal Rule tags are,

  • align="left|center|right": Specifies the alignment of the horizontal rule.
  • noshade: Removes the shading effect, making the line solid.
  • size="value": Specifies the height of the horizontal rule.
  • width="value|%": Specifies the width of the horizontal rule.
  • color="color": Specifies the color of the horizontal rule.

Break Line Tag

The break line tag is used to insert a line break in an HTML document. It is an empty tag and does not require a closing tag.

Syntax:

<br>

Comment in HTML

Comments are used to insert notes or explanations in the HTML code. Comments are not displayed in the browser.

Syntax:

<!-- This is a comment -->

Center Tag

The center tag is used to center-align text or other content in an HTML document. Note that the <center> tag is not supported in HTML5 and CSS should be used instead.

Syntax:

<center>This text is centered</center>

Paragraph Tag

The paragraph tag is used to define a paragraph in an HTML document. It has an opening tag <p> and a closing tag </p>.

Syntax:

<p align="center">This is a paragraph.</p>

Attributes of Paragraph tags are,

  • align="left|center|right|justify": Specifies the alignment of the paragraph.

Output:

This is a paragraph.

Font Tag

The font tag is used to define the font, size, and color of the text in an HTML document. It has an opening tag <font> and a closing tag </font>.

Syntax:

<font size="4" color="blue" face="Arial">This is a text.</font>

Attributes of Font tags are,

  • size="value": Specifies the size of the text.
  • color="color": Specifies the color of the text.
  • face="font": Specifies the font of the text.

Output:

This is a text.

HTML Entities

HTML entities are used to display reserved characters in HTML. For example, to display the less than sign (<), you can use the entity &lt;.

Syntax:

&lt; &gt; &amp; &quot; &apos;

Image Tag

The image tag is used to embed an image in an HTML document. It is an empty tag and does not require a closing tag.

Syntax:

<img src="url" alt="description" height="100" width="100" align="left" border="1">

Attributes of Image tags are,

  • src="url": Specifies the URL of the image.
  • alt="description": Provides an alternative text for the image.
  • height="value": Specifies the height of the image.
  • width="value": Specifies the width of the image.
  • align="left|right|top|middle|bottom": Specifies the alignment of the image.
  • border="value": Specifies the border width of the image.

Anchor Tag

The anchor tag is used to create hyperlinks in an HTML document. It has an opening tag <a> and a closing tag </a>.

Syntax:

<a href="url" name="anchorName">Link Text</a>

Attributes of Anchor tags are,

  • href="url": Specifies the URL of the link.
  • name="anchorName": Specifies the name of the anchor.

Output:

Example Link

Ordered List

The ordered list tag is used to create an ordered list in an HTML document. It has an opening tag <ol> and a closing tag </ol>.

Syntax:

<ol type="1" start="1">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Attributes of Ordered List tags are,

  • type="1|A|a|I|i": Specifies the type of numbering.
  • start="value": Specifies the starting number of the list.

Output:

  1. Item 1
  2. Item 2

Unordered List

The unordered list tag is used to create an unordered list in an HTML document. It has an opening tag <ul> and a closing tag </ul>.

Syntax:

<ul type="disc">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Attributes of Unordered List tags are,

  • type="disc|circle|square": Specifies the type of bullet points.

Output:

  • Item 1
  • Item 2

Nested List

A nested list is a list within another list.

Syntax:

<ul>
  <li>Item 1
    <ul>
      <li>Subitem 1</li>
      <li>Subitem 2</li>
    </ul>
  </li>
  <li>Item 2</li>
</ul>

Output:

  • Item 1
    • Subitem 1
    • Subitem 2
  • Item 2

Table

The table tag is used to create a table in an HTML document. It has an opening tag <table> and a closing tag </table>.

Syntax:

<table border="1" bordercolor="black" bgcolor="white" background="url" rules="all" frame="box" height="200" cellpadding="5" cellspacing="5">
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td align="center" width="100" height="50" bgcolor="lightgrey" rowspan="2" colspan="2" background="url">Data 1</td>
      <td>Data 2</td>
    </tr>
    <tr>
      <td>Data 3</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">Footer</td>
    </tr>
  </tfoot>
</table>

Attributes of Table tags are,

  • border="value": Specifies the border width of the table.
  • bordercolor="color": Specifies the border color of the table.
  • bgcolor="color": Specifies the background color of the table.
  • background="url": Specifies the background image of the table.
  • rules="none|groups|rows|cols|all": Specifies the rules for the table borders.
  • frame="void|above|below|hsides|lhs|rhs|vsides|box|border": Specifies which parts of the frame to display.
  • height="value": Specifies the height of the table.
  • cellpadding="value": Specifies the space between the cell content and its borders.
  • cellspacing="value": Specifies the space between the cells.

Attributes of TD tags are,

  • align="left|center|right": Specifies the alignment of the cell content.
  • width="value": Specifies the width of the cell.
  • height="value": Specifies the height of the cell.
  • bgcolor="color": Specifies the background color of the cell.
  • rowspan="value": Specifies the number of rows a cell should span.
  • colspan="value": Specifies the number of columns a cell should span.
  • background="url": Specifies the background image of the cell.

Output:

Header 1 Header 2
Data 1 Data 2
Data 3
Footer

Form Tag

The form tag is used to create a form in an HTML document. It has an opening tag <form> and a closing tag </form>.

Syntax:

<form name="formName" enctype="multipart/form-data" method="post" action="url">
  <input type="text" name="inputName" value="defaultValue" size="20" maxlength="30" align="left">
  <input type="checkbox" name="checkboxName" checked>
  <select name="selectName" multiple size="3">
    <option value="1" selected>Option 1</option>
    <option value="2">Option 2</option>
  </select>
  <textarea name="textareaName" cols="30" rows="10">Default text</textarea>
</form>

Attributes of Form tags are,

  • name="formName": Specifies the name of the form.
  • enctype="multipart/form-data": Specifies the encoding type of the form data.
  • method="get|post": Specifies the HTTP method to use when sending form data.
  • action="url": Specifies the URL to send the form data to.

Attributes of Input tags are,

  • name="inputName": Specifies the name of the input field.
  • checked: Specifies that the input field should be pre-selected.
  • value="defaultValue": Specifies the default value of the input field.
  • size="value": Specifies the width of the input field.
  • maxlength="value": Specifies the maximum number of characters allowed in the input field.
  • align="left|right|top|middle|bottom": Specifies the alignment of the input field.

Attributes of Select tags are,

  • name="selectName": Specifies the name of the select field.
  • multiple: Specifies that multiple options can be selected.
  • size="value": Specifies the number of visible options in the select field.

Attributes of Textarea tags are,

  • cols="value": Specifies the width of the textarea.
  • rows="value": Specifies the height of the textarea.
  • name="textareaName": Specifies the name of the textarea.

Output:
